# consolidated-bill

## binfen-base

缤纷基础脚手架,基于vant less eslint postcss-pxtorem svg-sprite-loader


组件封装:

svg-icon svg小图标

nav-bar-base 顶部导航
样式:

var.less 定义变量

theme.less 自定义主题

mixin.less 混合器

index.less 全局样式

其他样式可在index.less 中引入或者在组件中单独引入
